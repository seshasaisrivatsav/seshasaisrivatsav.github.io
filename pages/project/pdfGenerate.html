About

This JavaScript project allows users to dynamically create PDFs. With JavaScript objects either created locally or dynamically from API, you can create PDF. Internally pdfkit uses pdfmake to achieve this. They are in package.json

GitHub URL

Running the project

  1. Git clone project
  2. npm run
  3. Run the generatePDF.js
  4. Enjoy the nice PDFs generated
